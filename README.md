# Scoop Bucket for taws

This is a [Scoop](https://scoop.sh/) bucket for [taws](https://github.com/huseyinbabal/taws) - Terminal UI for AWS.

## Installation

```powershell
scoop bucket add huseyinbabal https://github.com/huseyinbabal/scoop-bucket
scoop install taws
```

## Update

```powershell
scoop update taws
```
